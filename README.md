# Travel Itinerary App

Welcome to the Travel Itinerary App! This application helps users plan, organize, and manage their travel itineraries with ease.

## Features
- User onboarding for a personalized experience
- Dashboard to view and manage trips
- Accommodation, transport, and activity cards
- Context-based state management
- Modern and responsive UI

## Getting Started

### Prerequisites
- Node.js (v14 or higher recommended)
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```bash
   cd project
   ```
3. Install dependencies:
   ```bash
   npm install
   # or
   yarn install
   ```

### Running the App
To start the development server:
```bash
npm start
# or
yarn start
```
By default, the app runs on [http://localhost:3000](http://localhost:3000).
